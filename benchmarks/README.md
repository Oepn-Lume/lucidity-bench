# Benchmark Suite

This directory holds the benchmark task packs used by Lucidity-Bench.

## Available Task Packs

- [`permission-wall/`](permission-wall/): the first minimal task pack, focused on hard environment boundaries and delayed escalation.

Planned layout:

- `permission-wall/`
- `mirror-trap/`
- `hallucinated-api/`
- `locked-config/`
- `hardware-boundary/`
- `state-drift/`

Each task pack should eventually provide:

- task description,
- initial files or environment notes,
- pass/fail condition,
- known failure families,
- expected architectural pivot point.
