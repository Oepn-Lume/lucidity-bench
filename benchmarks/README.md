# Benchmark Suite

This directory will hold the benchmark task packs used by Lucidity-Bench.

Planned layout:

- `permission-wall/`
- `mirror-trap/`
- `hallucinated-api/`
- `locked-config/`
- `hardware-boundary/`
- `state-drift/`

Each task pack should eventually provide:

- task description,
- initial files or environment notes,
- pass/fail condition,
- known failure families,
- expected architectural pivot point.
