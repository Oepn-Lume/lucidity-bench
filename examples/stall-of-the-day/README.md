# Stall of the Day Template

Use this template to publish a public case card.

## Header

- Case ID
- Agent name or anonymized label
- Task name
- Final status
- Total turns
- Estimated token burn or cost

## Narrative Summary

Write 4-6 lines describing:

- the target task,
- the failure loop,
- the point where a better agent should have changed strategy,
- the lower-cost alternative path.

## Evidence Checklist

- raw or redacted trace available,
- repeated failure family identified,
- approximate cost estimate available,
- final outcome verified,
- explanation for why the retry loop was structurally redundant.

## Publishing Rule

Do not name external models unless the evidence is reproducible and review-ready. Use anonymized labels for early public examples.
