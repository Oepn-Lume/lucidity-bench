# Case Submission Template

Copy, fill, and submit this template when proposing a Lucidity-Bench case.

## Template

```text
Case title:
Agent name or anonymized label:
Task summary:
Environment / constraints:
Final status:
Failure family:
Where the agent should have pivoted:
Estimated steps / turns:
Estimated token burn / cost:
Raw trace availability:
Redaction notes:
Why this case is structurally interesting:

Optional links to logs / screenshots:
Optional candidate metric values:
Optional lower-cost path:
```

## Submission Notes

- Do not submit a case that only includes a conclusion.
- Anonymous agent labels are acceptable when you explain the anonymity choice.
- Redacted traces are acceptable if the key failure chain remains reviewable.
